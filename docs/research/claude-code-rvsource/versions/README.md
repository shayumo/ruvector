## How to Use

```bash
# Rebuild all versions from scratch
node scripts/rebuild-all-versions.mjs

# Rebuild only specific series
node scripts/rebuild-all-versions.mjs --series 2.0,2.1

# Or use the shell wrapper
./scripts/claude-code-rvf-corpus.sh
```

## Format

Each version directory contains (source and RVF cleanly separated):
- `source/` - Decompiled JavaScript modules organized by category
  - `core/`, `tools/`, `permissions/`, `config/`, `telemetry/`, `ui/`, `types/`, `uncategorized/`
  - `witness.json` - SHA-256 witness chain for provenance
- `rvf/` - Binary RVF containers (128-dim cosine-distance HNSW index)
  - `master.rvf` - All vectors combined
  - Per-category RVFs: `core.rvf`, `tools.rvf`, `permissions.rvf`, etc.
  - Each `.rvf` has a `.manifest.json` sidecar with vector-to-fragment mapping
- `metrics.json` - Code metrics and module statistics

Generated by `scripts/rebuild-all-versions.mjs` using the decompiler library
at `npm/packages/ruvector/src/decompiler/`.
